Session 3: Classification

You have 4 classifiers to test out.

Each classifier will work with you for a series of leaves. They make a recommendation for which class a leaf belongs to. Diffierent classifiers work in different ways. The same classifier works in the same way for a series.

All the classifiers make a correct recommendation about 70-90% of the time. Some might be better; some might be worse. The goal here is to make a good decision -- follow the classifier when it is correct, and do not follow the classifier when it is incorrect.
