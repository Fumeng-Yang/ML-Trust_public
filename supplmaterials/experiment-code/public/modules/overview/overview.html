Overview

Do you trust a classifier?

A classifier is a person or a system that uses an object's characteristics to identify which category the object belongs to. You will work with automated machine classifiers to identify objects and decide whether to trust the machine classifiers.

We use the following definition of trust. In our experiment, the automated system will be a classifier.
Trust consists of two components; one is a general attitude towards the performance of an automated system and another is the willingness at a given point in time to accept a recommendation from the automated system.

Today's experiment has 4 sessions in total.
I. a questionnaire about your background and experience;
II. a session introducing background information;
III. a session where you work with several classifiers;
IV. a questionnaire about your opinion on this experiment.

You will see detailed instructions in each session. If you have any questions regarding the experiment, feel free to ask the experimenter(s).
